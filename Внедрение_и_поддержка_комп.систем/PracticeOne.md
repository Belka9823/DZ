# Checklist for Medical Mask Testing (QA)

| Категория | Вид тестирования | Объект проверки | Ожидаемый результат |
| :--- | :--- | :--- | :--- |
| Основы | Functional | Проверка фильтрации (BFE) | Задерживает >95% бактерий и микрокапель |
| | Functional | Тест носового фиксатора | Гнется под форму носа и держит изгиб |
| | Functional | Дыхательный тест | Сопротивление дыханию минимально |
| Нагрузка | Stress | Натяжение ушных петель | Место спайки выдерживает нагрузку до 10Н |
| | Stress | Тест на горение | Материал плавится, но не поддерживает открытое горение |
| Негатив | Negative | Проверка влагостойкости | Внешний слой отталкивает капли жидкости |
| | Negative | Тест на износ | Структура фильтра разрушается при намокании |
| Безопасность| Security | Гипоаллергенность | Отсутствие ворса и раздражения на коже |
| | Security | Запах | Отсутствие химического запаха при вскрытии |
| Визуал | UI Testing | Ориентация складок | Складки на внешней стороне направлены вниз |
| | UI Testing | Качество швов | Ультразвуковая пайка без пропусков и дыр |
| Среда | Compatibility | Работа с очками | Отсутствие запотевания при правильной фиксации |
| Документы | Documentation | Инструкция | Наличие правил утилизации на упаковке |
